# vcpkg

Vcpkg 是微软 C++ 团队开发适用于Windows，Linux 和 MacOS 的 C 和 C++ 库管理工具。
特点:
1. 支持多平台
2. 支持多triplet
3. 支持依赖库自动安装
4. 支持port导出功能
